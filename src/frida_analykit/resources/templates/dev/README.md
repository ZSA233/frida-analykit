# Dev Workspace

Generated by `frida-analykit` `${version}`.

## Quick Start

1. Install dependencies:

   `npm install`

   The generated runtime dependency comes from npmjs, so no `.npmrc` or extra token is required.

2. Edit `index.ts` and keep `config.yml` pointing at `_agent.js`.

3. Let the Python CLI trigger the compile flow:

   `frida-analykit build --config config.yml`

   `frida-analykit attach --config config.yml --build --repl`

   `frida-analykit attach --config config.yml --watch --repl`

4. Manage the remote `frida-server` from the same config when needed:

   `frida-analykit doctor --config config.yml`

   `frida-analykit server install --config config.yml`

   Set `server.version` in `config.yml` or pass `--version` to pin a specific `frida-server` build.

5. The npm scripts still exist for advanced workflows, and the CLI reuses them:

   `npm run build`

   `npm run watch`
